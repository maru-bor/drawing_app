# SimpleART

SimpleART is a desktop drawing application for Windows built with C#, WPF and SkiaSharp mainly aimed at beginner artists and anyone who prefers simplified drawing software. It provides all the basic digital drawing features listed below. It aims to be the middle ground between very simple and overly complex drawing applications. 

---

## Main features
- Drawing on a canvas
- Creating a new canvas
- Undo/Redo
- Setting brush properties (thickness, opacity, color, spacing)
- Importing custom brushes
- Drawing on different layers
- Managing layers (adding, rearranging, deleting)
- Exporting drawings

---

## Technologies Used
- C# / .NET 10
- WPF
- SkiaSharp

---

## Installation and setup guide
1. Go to the [Releases page](https://github.com/maru-bor/drawing_app/releases/tag/v1.0.0).
2. Download **`SimpleART-v1.0.0-win-x64.zip`**.
3. Extract the ZIP file to any folder.
4. Open the **`publish`** folder.
5. Find and run **`SimpleART.exe`**.
> **Note:** Windows may show a SmartScreen warning on first launch because the executable is not code-signed. Click **More info → Run anyway** to proceed.
